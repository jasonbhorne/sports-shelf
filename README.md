# The Sports Desk

Source for the Sports page at https://www.jasonhorne.org/sports, served via
GitHub Pages and embedded in Squarespace with an auto-resizing iframe.

Card links:

- ABS Challenge Tracker (live dashboard)
- Braves True-Talent Dashboard (live dashboard)
- March Madness bracket predictor (ML model)
- AI Fantasy Football Assistant (Claude Code skill)

To add a card, edit `index.html` (one `<article class="card">` per project,
in the right group) and push. The Squarespace page updates on the next visit.

Built with Claude Code.
